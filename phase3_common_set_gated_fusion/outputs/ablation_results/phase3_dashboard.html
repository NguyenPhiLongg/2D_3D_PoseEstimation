How to Interpret This Dashboard

1. Fair comparison

Phase 3 is designed to compare all models on the same common video subset. This avoids unfair comparison between the old 2D baseline and 3D/Fusion models.

2. Concat Fusion

Concat Fusion directly joins 2D features and 3D features: 40D + 59D = 99D. This is a fixed early-fusion strategy.

3. Gated Fusion

Gated Fusion learns how much the model should rely on 2D or estimated 3D features. It is useful because estimated 3D pose may contain errors from YOLOv8-Pose and PoseFormerV2.

4. Important metric for fall detection

Fall Recall is especially important because missing a real fall can be more serious than raising a false alarm.
